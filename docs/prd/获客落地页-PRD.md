## 2. 产品定位与目标

获客落地页面向访客提供街道/乡镇维度的潜在高企分析预览，并通过名单解锁、免费演示申请等动作收集访客线索。

目标：

- 支持访客选择辖区街道或乡镇后查看分析结果。
- 通过模糊名单预览引导访客提交姓名、单位、手机号以解锁完整名单。
- 支持访客提交免费演示申请，形成后续顾问联系线索。

## 6. 核心业务流程

### 6.1 街道分析与名单解锁流程

1. 访客进入获客落地页。
2. 访客选择辖区街道或乡镇。
3. 访客点击“立即分析”。
4. 页面展示所选街道/乡镇的分析结果，包括企业总数、泛科技企业、高潜力企业和前三条预览企业。
5. 未解锁时，企业名单行以模糊形式展示。
6. 访客点击解锁完整名单，并填写姓名、单位、手机号。
7. 提交成功后，页面显示解锁成功和顾问联系提示。
8. 留资后状态变为已提交。

### 6.2 免费演示申请流程

1. 访客进入演示申请区域。
2. 访客填写姓名、所在单位、手机号。
3. 访客提交申请。
4. 页面展示申请已提交。
5. 提交后不重复展示表单。
