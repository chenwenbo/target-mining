## 2. 产品定位与目标

运营后台面向平台运营人员，用于进入平台管理控制台，查看平台级经营与使用概览，管理租户基础信息、租户账号状态、管理员凭证和账号开通相关导出/复制操作。

产品目标：

| 目标 | 说明 |
| --- | --- |
| 平台运营可视化 | 提供平台 KPI、趋势图、分布图、活跃租户排行，支撑运营人员快速掌握平台使用情况。 |
| 租户全生命周期管理 | 支持租户创建、查询、筛选、编辑、启用/禁用、详情查看与备注维护。 |
| 账号开通与凭证交付 | 支持租户管理员账号、密码查看、重置、复制、批量复制与 CSV 导出。 |

## 6. 核心业务流程

### 6.1 运营登录流程

1. 平台运营输入运营账号和密码。
2. 系统校验账号密码。
3. 校验成功后进入 `/ops/dashboard`。
4. 校验失败时展示“账号或密码错误”。

### 6.2 租户管理流程

1. 平台运营进入租户管理列表。
2. 可按租户名、行政区、账号搜索，或按状态 Tab 筛选。
3. 可新增租户，录入租户基础信息、管理员账号、联系人信息。
4. 可编辑租户基础信息、管理员信息、联系人信息和备注。
5. 可启用或禁用租户账号。
6. 可从列表点击租户名进入租户详情。

### 6.3 租户账号凭证流程

1. 平台运营进入租户详情或账号开通页面。
2. 密码默认隐藏，可按需显示。
3. 可二次确认后重置租户管理员密码。
4. 可复制单个密码、复制单个账号凭证、批量复制当前筛选结果的账号凭证。
5. 可导出当前筛选账号列表为 CSV。

## 8. 权限与数据规则

### 8.1 权限规则

| 规则 | 来源/状态 |
| --- | --- |
| 平台运营可访问运营后台全部功能。 | 来源于功能清单适用角色。 |

### 8.2 数据与交互规则

| 规则类型 | 规则 |
| --- | --- |
| 概览数据 | 平台 KPI 由租户 mock 数据聚合；月度新增租户趋势展示近 6 个月；企业分布以各租户企业数量占比展示；租户活跃度排行按活跃度排序取前 6。 |
| 租户状态 | 支持全部、活跃、试用中、已到期、已禁用筛选；禁用后状态改为已禁用，启用后恢复活跃；详情页状态需与租户列表保持一致。 |
| 搜索分页 | 租户搜索支持按租户名、行政区、账号搜索；搜索后重置到第一页；租户列表超过分页大小时支持上一页/下一页，并展示页码和总页数。 |
| 租户创建 | 租户名称、城市、行政区、到期日、用户名必填；密码自动生成且可改。 |
| 租户编辑 | 编辑租户基础信息、管理员信息、联系人信息、备注；账号名和密码在详情页管理。 |
| 密码展示 | 租户详情和账号凭证列表中的密码默认隐藏，可单独显示或隐藏。 |
| 密码重置 | 首次点击进入确认态，二次确认后生成新密码并展示。 |
| 复制凭证 | 单密码复制后图标切换；单账号凭证格式为用户名和密码；全部复制包含当前筛选结果中的租户名、用户名、密码。 |
| CSV 导出 | 导出当前筛选账号列表，字段包含租户名、行政区、状态、用户名、密码、到期日，且中文正常。 |
